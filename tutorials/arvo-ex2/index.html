ARVO Demo 2 — Quick analysis on GRAPE

This exercise reproduces Exercise 2 from the ARVO AI4Oph repository. It demonstrates simple exploratory data analysis and linear regression on the GRAPE dataset using pandas, matplotlib, and scikit-learn.

Open in Colab View on GitHub Original Instructions

About this demo

The GRAPE dataset contains measurements of retinal biomarkers from a small synthetic cohort. You will explore basic relationships between predictors and outcomes, create quick plots, and fit a linear regression model.

Example workflow

df.head()
  2. Explore summary statistics and missing data:
    3. df.describe()
df.isna().mean()
  3. Fit a quick regression and plot:
    4. m = LinearRegression().fit(df[['Predictor']], df['Outcome'])
plt.scatter(df['Predictor'], df['Outcome'])
plt.plot(df['Predictor'], m.predict(df[['Predictor']]), color='r')
