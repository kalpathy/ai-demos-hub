ARVO Demo 1 — Table extraction from screenshots

Use ChatGPT to extract table data from a screenshot and download CSV. Two example images are provided.

Repo Instructions

Assets

Download one of these images, then upload it to ChatGPT and prompt:

“Can you extract the data in the provided screenshot and return it in CSV format?”
  • iop_table.png
  • GRAPE_screenshot.png
Demo flow
  1. Download one of the images above.
  2. In ChatGPT, start a new chat and upload the image.
  3. Paste the prompt above; request a CSV download link.
  4. Optionally ask follow-ups like column renaming or basic statistics.

Source: ARVO25_AI4Oph Exercise 1 (screenshots & instructions).
