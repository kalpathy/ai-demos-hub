ARVO Demo 3 — NotebookLM over papers

Build a lightweight literature “notebook” using NotebookLM: upload a small set of PDFs and ask focused questions for summarization, comparisons, and quick figures.

Open NotebookLM Original Instructions Sample Papers (repo)

Quick start

  1. Open NotebookLM and create a new notebook.
  2. Add sources:
    • Upload PDFs (e.g., from exercise_3/papers in the ARVO repo), or
    • Paste stable public URLs (publisher PDFs or raw GitHub PDF links).
  3. Use guided prompts below to generate summaries, compare methods, and extract key tables.

Guided prompts (copy/paste)

  • Summarize each paper in two bullet points: main goal and primary result.
  • Create a comparison table with columns: Paper, Modality, Dataset size, Primary metric, Best value, Limitations.
  • Extract all reported AUC values and list them with 95% CIs if available.
  • From the methods sections, list preprocessing steps and model classes used.
  • Provide a 5-sentence synthesis: where do these papers agree or disagree, and why?
Tips
  • Use 2–4 PDFs for a crisp live demo.
  • If a PDF fails to parse, upload the file instead of a URL.
  • Ask for tables; NotebookLM can output tabular summaries you can copy to CSV.
